Welcome to HTML Tags Explanation Page

This page is built to practice and explain basic HTML tags. Let's start learning!

1. Paragraph Tag & Text Formatting Tags

The <p> tag is used to create a paragraph.
You can make text bold using the <strong> tag or bold using the <b> tag.
You can underline text with the <u> tag.
Use italic text with the <i> tag or emphasize text with the <em> tag.
You can highlight text using the <mark> tag.
To show a short quotation, use the quotation tag.
To show crossed-out text, use the strikethrough tag.

2. Image Tag

Use the <img> tag to display an image:


The attributes that an img tag can contain are:

  • The src attribute sets the image path
  • title shows a tooltip
  • alt provides alternative text if the image doesn't load'
  • width sets the image with
  • height sets the image height

3. Link Tag

The <a href=" link " target="" > tag creates a hyperlink. Example: google
The href sets the link's destination, and target="_blank" opens it in a new tab.

4. Input Fields

HTML forms use different <input type="Input_type"/> types:

  • Password input
  • Disabled text input
  • Number input
  • Email input
  • Color picker
  • Upload a file
  • Telephone number input
  • Date picker
  • Date and time picker
  • Radio button (choose one option)
  • Checkbox (choose multiple options)
  • Range slider
